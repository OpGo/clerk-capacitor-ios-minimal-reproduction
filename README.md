# clerk-capacitor-minimal-reproduction


## Setup

Create `.env.local` with


```
VITE_CLERK_PUBLISHABLE_KEY=...
```


```
npm install
```


## Web

```
npm run start
```




## Capacitor

```
npm run build-app-sync
```


### iOS

```
npx cap open ios
```

Hit play in Xcode.



### Android

```
npx cap open android
```

Hit play in Android Studio.
